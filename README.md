### 源码调试



```

* 首先，请先在本地visualstudio内建立项目(每个源码对应一个小项目) 

* 然后，将代码源文件拷贝到项目内建立的源文件中;

* 最后，F5进行编译和运行，F10单步快速调试，F11步进式调试，F12查看源码定义

```
